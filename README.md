# Red Dragons 

![](http://i.imgur.com/FwIahcm.png)

Red Dragons é a equipe de futebol de robô da Universidade Federal de São Carlos, criada em 2013 com o apoio dos departamento de engenharia elétrica e engenharia mecânica. Para mais informações e contato, acesse o nosso [site](http://www.robotica.ufscar.br/~robotica/red/)

## Controle

Este repositório hospeda os códigos responsáveis pelo controle dos nossos robôs, e está aberto para consulta geral. 

[22/05/2017] O código ainda está em desenvolvimento, e para o funcionamento do mesmo é imprescindível o alinhamento com o código de visão, que pode ser encontrado aqui, mas não é atualizado e mantido por esse diretório.

O uso do mesmo deve ser referenciado e qualquer melhoria será muito bem vinda à nossa equipe!

Equipe Red Dragons.
